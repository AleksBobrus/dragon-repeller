# 🐉 Dragon Repeller – RPG-игра в браузере

[![Demo](https://img.shields.io/badge/Demo-Live-brightgreen?style=flat&logo=github)](https://aleksbobrus.github.io/dragon-repeller/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Текстовая RPG, созданная по мотивам проекта freeCodeCamp.  
Игроку предстоит сразиться с драконом, который терроризирует город: покупать оружие, здоровье, сражаться с монстрами и искать секреты.

---

## 📋 Функционал

- **Три основные локации:** город, магазин, пещера.
- **Боевая система:** атака, уклонение, случайный урон, поломка оружия.
- **Магазин:** покупка здоровья и оружия, продажа ненужного снаряжения.
- **Противники:** слайм, клыкастый зверь и финальный босс – дракон.
- **Пасхальное яйцо:** секретная мини-игра с угадыванием чисел.
- **Адаптивный интерфейс:** работает на десктопе и мобильных устройствах.

---

## 🛠️ Технологии

- **HTML5** (семантическая вёрстка, ARIA-атрибуты)
- **CSS3** (Flexbox, кастомные свойства, медиа-запросы, градиенты)
- **JavaScript (ES6)** – манипуляции с DOM, объекты, массивы, обработчики событий

## 🚀 Быстрый старт

### Играть онлайн
Открой [живую демо-версию](https://aleksbobrus.github.io/dragon-repeller/) на GitHub Pages.

### Локальный запуск
1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/AleksBobrus/dragon-repeller.git
   ```
2. Открой файл index.html в любом браузере.

dragon-repeller/
